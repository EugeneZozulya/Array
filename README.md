# Array
This programm allows to do such operations with one-dimensional array:
- generate one-dimensional array in the range [-2000; 2000];
- save generated array in a file;
- download one-dimensional array from a file;
- sort generated array by descend;
- calculate sum of the array elements that are greater or less than the number and their index is even(user need input this number using special graphics element;
- calculate sum of the array elements and their count that are not in a range(user need input this range using special graphics element)
- calculate sum of the array elements which have a number composed of the first two digits is divisible by 5.
